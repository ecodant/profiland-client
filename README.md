## Egg's Adventure Landing Page
### Overview

Lading Page repository of Egg's Adventure, this project it's built with React, TypeScript, and Tailwind CSS.

### Technologies

- **React**
- **TypeScript**
- **Tailwind CSS**

### Initialization Steps

1. Install dependencies

   ```bash
   npm install
   ```
2. Start client server

   ```bash
   npm run dev
   ```
